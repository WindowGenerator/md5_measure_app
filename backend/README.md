## Installation:

- Install from source:
```bash
poetry install
```

## Development:

- Install dependencies:
```bash
make -f Makefile install-deps
```
- Run tests:
```bash
make -f Makefile test-unit
```
- Run pre-commit:
```bash
make -f Makefile pre-commit
```
- Run all:
```bash
make -f Makefile
```
